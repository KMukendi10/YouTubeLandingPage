# Kazadi YouTube Clone

A responsive YouTube homepage clone built using HTML and CSS, with additional interactive features implemented using JavaScript. The project recreates the look and feel of YouTube's homepage, including a navigation bar, sidebar, filter buttons, and a responsive video grid.

## Features

### Core Features

* Responsive YouTube-inspired homepage layout
* Navigation bar with search functionality
* Sidebar navigation menu
* Video grid layout
* Category filter buttons
* Embedded YouTube videos
* Mobile-friendly design

### Custom Features

#### Feature Built Without Cursor

* **Dark/Light Mode Toggle**

  * Allows users to switch between light mode and dark mode.
  * Implemented manually using JavaScript and CSS.

#### Features Built With Cursor

* **Video Search**

  * Filters videos based on the user's search input.

* **Video Card Hover Effect**

  * Highlights video cards with a background color when hovered over to improve user interaction.
